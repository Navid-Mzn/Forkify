# Forkify

A modern recipe search application built with JavaScript.

## About the Project

Forkify is a recipe application developed as part of Jonas Schmedtmann's JavaScript course. Users can search for recipes, view detailed cooking instructions, adjust serving sizes, bookmark favorite recipes, and upload their own recipes.

The project demonstrates modern JavaScript development using modules, asynchronous programming, API integration, and the MVC architecture.

## Features

- Search for thousands of recipes
- View detailed recipe information
- Adjust servings automatically
- Bookmark favorite recipes
- Upload custom recipes
- Responsive user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Parcel Bundler
- MVC Architecture
- REST API
- Async/Await

## What I Learned
